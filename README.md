# E-Commerce Performance & Customer Experience Dashboard

## 📊 Overview
This project analyzes e-commerce data to understand customer behavior, delivery performance, and seller impact using Power BI.

## 🔑 Key Insights
- Delivery performance is the main driver of customer satisfaction
- Very slow deliveries (>20 days) significantly reduce ratings
- Revenue is highly concentrated among top customers (~47%) and sellers (~67%)
- Some high-revenue sellers underperform in customer experience
- Regional differences indicate operational inefficiencies

## 📈 Dashboard Pages
1. Business Overview  
2. Delivery & Rating Analysis  
3. Seller & Regional Analysis  
4. Final Insights & Recommendations  

## 🛠 Tools Used
- Power BI  
- DAX  
- Data Cleaning & Analysis
